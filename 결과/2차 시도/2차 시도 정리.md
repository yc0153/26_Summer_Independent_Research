# 2차 시도 정리

1차 시도에서 얻은 결과를 바탕으로, 2차 시도에서는 다음 두 가지를 먼저 정리했다.

1. 기본 네 점을 조금 더 바깥쪽으로 넓혔다.

   - 철근망 전체를 좀 더 안정적으로 포함시키도록 코너 기본값을 다시 잡았다.

2. warp 이후와 affine rectification 이후에 결과 이미지를 한 번 더 정리했다.

   - 검은 바깥 영역과 불필요한 가장자리가 counting에 덜 영향을 주도록 중간 결과를 분리했다.
   - contour는 counting의 핵심이 아니라, 전체 흐름을 설명하는 보조 도구로만 취급했다.

## 현재 결과

`target_image.jpg` 기준 2차 시도 결과는 다음과 같다.

- x축 방향: 5개
- y축 방향: 12개

## 2차 시도에서 보인 점

1. 결과 이미지가 1차보다 더 정돈되어 보인다.

2. 다만 가장자리 쪽 철근은 아직 임계값과 겹쳐서 일부가 합쳐지거나 따로 잡힌다.

3. 다음 시도에서는 threshold를 조금 더 세밀하게 조정하거나, line cluster 기반 카운팅을 추가로 실험하는 쪽이 좋다.
